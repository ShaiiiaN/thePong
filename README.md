## Getting Started

1. Clone the repo

   ```sh
   git clone https://github.com/ShaiiiaN/thePong.git
   ```

2. Go to the server directory

   ```sh
   cd server/
   ```

3. Install NPM packages
   ```sh
   npm i
   ```
4. Set Port in `.env`
   ```js
   PORT = "8000";
   ```
5. Run the server
   ```sh
   npm start
   ```
